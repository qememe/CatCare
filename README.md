# CatCare

**CatCare** — это графическое приложение на C# Windows Forms для помощи в уходе за домашним котом.

Приложение показывает карточку кота, позволяет отредактировать базовые данные питомца и помогает отслеживать кормление и внимание к коту.

## Цель проекта

Цель проекта — создать удобное приложение-помощник для владельца кота, которое помогает не забывать о базовом уходе за питомцем.

## Основные возможности

- указание клички;
- указание даты рождения;
- указание даты принятия кота в семью;
- настройка периода кормления;
- настройка периода поглаживания;
- отображение возраста кота;
- отображение времени, которое кот живёт с владельцем;
- отображение времени до следующего кормления;
- отображение времени до следующего поглаживания;
- отметки "Кормил" и "Гладил".

## Технологии

- язык программирования: **C#**
- платформа: **.NET Framework**
- интерфейс: **Windows Forms**
- хранение данных: **обычные переменные в коде**

## Как работает приложение

В проекте данные кота записаны прямо в коде:

- кличка;
- дата рождения;
- дата принятия в семью;
- период кормления;
- период поглаживания.

Файлы для хранения не используются. После закрытия программы изменённые данные и новые отметки не сохраняются, потому что это упрощённая учебная версия.

При запуске открывается окно профиля, где можно изменить имя кота, дату рождения и дату принятия в семью. На главном экране отображается информация о питомце, возраст, время жизни с владельцем, время до следующего кормления и время до следующего поглаживания.

## Реализованный функционал

- главное окно с информацией о коте;
- редактирование профиля кота;
- кнопка “Кормил”;
- кнопка “Гладил”;
- автоматический расчёт следующего кормления;
- автоматический расчёт следующего поглаживания;
- сообщения после отметок о кормлении и поглаживании;
- выбор отображения времени в минутах, часах или днях.

## Ограничения

- данные хранятся только во время работы программы;
- после перезапуска приложения профиль и отметки возвращаются к значениям из кода;
- загрузка настоящей фотографии кота пока не реализована.
