# CatCare

**CatCare** — это графическое приложение на C# Windows Forms для помощи в уходе за домашним котом.

Приложение позволяет создать профиль питомца, сохранить основные данные о нём и получать напоминания о кормлении и взаимодействии с котом.

## Цель проекта

Цель проекта — создать удобное приложение-помощник для владельца кота, которое помогает не забывать о базовом уходе за питомцем.

## Основные возможности

- создание профиля кота;
- указание клички;
- указание даты рождения;
- указание даты принятия кота в семью;
- настройка периода кормления;
- отображение возраста кота;
- отображение времени, которое кот живёт с владельцем;
- напоминание о кормлении;
- случайные напоминания об уходе и внимании;
- сохранение настроек в JSON-файл.

## Технологии

- язык программирования: **C#**
- платформа: **.NET Framework**
- интерфейс: **Windows Forms**
- хранение данных: **JSON**

## Как работает приложение

При первом запуске пользователь заполняет данные о своём коте:

- кличка;
- дата рождения;
- дата принятия в семью;
- период кормления;
- фотография кота.

После сохранения данные записываются в JSON-файл.  
При следующих запусках приложение автоматически загружает сохранённый профиль.

На главном экране отображается информация о питомце, возраст, время жизни с владельцем и время следующего кормления.

## Планируемый функционал

- главное окно с информацией о коте;
- окно первичной настройки;
- редактирование профиля кота;
- кнопка “Покормил”;
- автоматический расчёт следующего кормления;
- всплывающие уведомления;
- случайные советы по уходу;
- возможность сбросить данные.
